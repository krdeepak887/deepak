Projects

Project - 1, "Computing Server".

IDEAS - Technology Innovation Hub.
A server is a system of computation, which provides centralised services to a network of computers. Computers that access the services of a server are called clients. Networks using servers are a kind of client-server, used in networks of medium and large machines (and in networks where the issue of security plays a major role). The term is widely applied to complete computers, but a server may equate to software or parts of a computer system, or even a machine, which is not necessarily a computer.
Software used - CUDA,NVIDIA GPU,Python,TensorFlow
Operating System - Linux
